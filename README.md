# apbd-cw2-oop-s34233
This is my simple C# console app.
The project is about renting university equipment.
User can rent equipment, return it, and check what is available.
Program can also show overdue rentals.
I was making this project in parts.
First I added structure and main classes.
Then I added services and UI.
Later I finished main program logic.
In the end I also changed some bool values into enums.
I also used branches when working on features.
This project was made for OOP practice and learning.
It is not very big, but it works.
